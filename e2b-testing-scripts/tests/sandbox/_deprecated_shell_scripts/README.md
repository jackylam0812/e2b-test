# 已废弃的 Shell 测试脚本

这些脚本已被新的 Python 自动化测试脚本取代。

## 迁移说明

### 旧方式（手动）
```bash
# 1. 手动创建沙箱
e2b sbx create

# 2. 手动进入沙箱
e2b sbx ssh <sandbox-id>

# 3. 在沙箱内手动执行
bash run_cpu_test.sh

# 4. 手动复制结果并退出
```

### 新方式（自动化）
```bash
# 一条命令完成所有操作
python3 ../run_sandbox_tests.py --test cpu
```

## 新脚本的优势

✅ **全自动化**：无需手动操作
✅ **实时输出**：测试过程中实时显示
✅ **结果管理**：自动保存到本地
✅ **错误处理**：自动捕获异常
✅ **易于集成**：可用于 CI/CD

## 相关文件

- `../run_sandbox_tests.py` - 新的自动化测试脚本
- `../README_AUTO_TESTS.md` - 详细使用文档

## 保留原因

这些脚本仅作为备份保留，以供参考。如果自动化脚本出现问题，可以参考这些原始脚本的逻辑。

**建议**：如果自动化测试稳定运行 1-2 周后，可以安全删除这些备份文件。
